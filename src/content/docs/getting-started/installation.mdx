### Requirements

<Steps>
  1. Skyrim Special Edition (supports SE and AE)
  2. [Skyrim Script Extender (SKSE)](https://www.nexusmods.com/skyrimspecialedition/mods/30379)
  3. [Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444)
  4. [Media Keys Fix SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/92948) to enable keyboard input
</Steps>

### Installation

<Tabs>
  <TabItem label="1.2.0 and above">
    <div className='flex flex-col gap-5'>
      <Steps>
        1. <VersionSelector client:load />
        2. Install the Prisma UI archive using your mod manager or manually to the `Skyrim/Data/` folder.
        3. OPTIONAL: If you are a developer then download the [PrismaUI_API.h](https://github.com/PrismaUI-SKSE/framework/releases/download/1.2.2/PrismaUI_API.h) header file for your own plugin.
      </Steps>
      <NexusLink className='w-[200px]' />
    </div>
  </TabItem>
  <TabItem label="1.1.0 and below">
    <Steps>
      1. <VersionSelector client:load />
      2. Put all files from `PUT FILES IN SKYRIM FOLDER` folder to your Skyrim game root directory
      3. Install all archives from `INSTALL WITH MO2` folder with your mod manager (Mod Organizer 2 or Vortex)
      4. If you are a developer, check `API HEADER FILE` folder for C++ header API file (see [Basic Usage](/getting-started/basic-usage/)). Debug version is available in `DEBUG VERSION` folder.
    </Steps>
  </TabItem>
</Tabs>

### Log File Location

If you encounter issues, check these log file for error messages:

- **SKSE Log:** `Documents\My Games\Skyrim Special Edition\SKSE\PrismaUI.log`
