Here's a list of what's planned for future releases:

- GPU rendering (improves performance for heavy CSS effects like blur/shadows/filters/gradients)
- New fast communication methods: `InteropCallString`, `InteropCallBool`, `InteropCallNumber`, `InteropCallVec2`, `InteropCallVec3`, `InteropCallVec4`
- Remove `Media Keys Fix SKSE` from dependencies
- MCP server for AI agents
- Gamepad support
- VR support
- Papyrus API
