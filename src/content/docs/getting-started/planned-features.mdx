We are constantly working on improving Prisma UI and adding new functionalities. Here's a list of what's planned for future releases:

- New `CreateAcceleratedView` method with GPU rendering (improves performance for heavy CSS effects like blur/shadows/filters/gradients)
- New fast communication methods: `InteropCallString`, `InteropCallBool`, `InteropCallNumber`, `InteropCallVec2`, `InteropCallVec3`, `InteropCallVec4`
- Remove Media Keys Fix SKSE from dependencies (own solution)
- MCP server for AI agents
- Gamepad support
- VR support
- Papyrus API
